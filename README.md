# 3D Graphics Programming Assignment
OpenGL project of a cartoonish cat in a house environment. 

## Controls
Press:
1: Toggle environmental lighting
2: Toggle indoor lighting
3: Toggle cartoon shader
4: Toggle camera attachment to the cat

## Usage
To run the project, open the solution and run the main.cpp file

## Authors and acknowledgment
